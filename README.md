# Selenium

Testes Automatizados com Selenium

O script de teste desenvolvido para a loja virtual de demonstraçao disponível em [VirtueMart](http://demo.virtuemart.net/).

Conceitos de automação, tais como:

-   TestSuites
-   TestCases
-   Tasks
-   AppObjects
-   Framework
-   Reports
-   Datapools
-   Dataprovider
